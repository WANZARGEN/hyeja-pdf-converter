# Hyeja Pdf Converter

Hyeja Pdf Converter Server

## Requirements

- React ^16.8
- Apollo Client

## Set up project

### Dotenv

```env
NODE_PATH=src/
SKIP_PREFLIGHT_CHECK=true
```

### How to install

- Change directory with `package.json`

```bash
$ npm install
```

## Running server

```bash
$ npm run start
```
